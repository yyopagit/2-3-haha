# Инструмент для поиска соседних провинций

## Описание

Скрипт `find_neighbors_final.py` анализирует карту `provinces.bmp` и находит ВСЕХ соседних провинций для указанных провинций, затем автоматически добавляет недостающие непроходимые границы в `adjacencies.csv`.

## Использование

```bash
cd "C:\Games\Vic2LV2\Victoria 2\mod\7"
python find_neighbors_final.py
```

## Что делает скрипт

1. Загружает `definition.csv` для получения RGB цветов провинций
2. Загружает карту `provinces.bmp`
3. Анализирует карту и находит всех соседей для каждой указанной провинции
4. Проверяет существующие непроходимые границы в `adjacencies.csv`
5. Генерирует недостающие непроходимые границы
6. Добавляет новые границы в `adjacencies.csv`

## Провинции для обработки

Скрипт обрабатывает следующие провинции:
- 1740, 1738, 1739 (Ливия)
- 1758 (Египет)
- 1719, 1724, 1718 (Алжир)
- 1594, 1598 (Синьцзян)
- 2094 (Южная Африка)
- 2077, 2074 (Ботсвана)
- 1161, 2586 (Саудовская Аравия)
- 934, 931, 929 (Ирак)

## Логи

Скрипт создает файл `neighbors_log.txt` с подробной информацией о процессе выполнения.

## Требования

- Python 3.x
- Pillow (PIL): `pip install Pillow`

## Примечания

- Анализ карты может занять некоторое время (несколько минут)
- Скрипт использует шаг 3 пикселя для баланса скорости и точности
- Все новые границы добавляются автоматически в правильное место в `adjacencies.csv`
