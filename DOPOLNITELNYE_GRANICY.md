# ДОПОЛНИТЕЛЬНЫЕ НЕПРОХОДИМЫЕ ГРАНИЦЫ

## Проблема

В `adjacencies.csv` для каждой провинции добавлено только по 2 непроходимые границы, но у каждой провинции обычно **4-7 соседей**.

## Текущее состояние

Для каждой из 17 провинций добавлено по 2 impassable границы:
- Например: 1740 → 1739, 1741
- Например: 929 → 930, 928

## Решение

Нужно найти **ВСЕХ** соседей для каждой провинции через анализ карты `provinces.bmp` и добавить impassable для всех найденных соседей.

### Инструмент для поиска всех соседей

Используйте скрипт `find_neighbors_quick.py`, который:
1. Анализирует `map/provinces.bmp` 
2. Находит всех соседей для каждой провинции
3. Автоматически добавляет недостающие impassable границы в `adjacencies.csv`

### Альтернативный подход

Можно вручную найти всех соседей для каждой провинции, используя:
- Карту игры
- Инструменты анализа карты
- Или анализ `definition.csv` и `provinces.bmp`
